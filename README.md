# Stay Balanced 💰

**Gestor de gastos y equilibrio financiero para estudiantes**

Aplicación móvil diseñada para ayudar a estudiantes universitarios a tomar mejores decisiones financieras, controlar sus gastos, ahorrar de forma inteligente y mantener el equilibrio entre sus ingresos y consumo.

---

## ✨ Funcionalidades Principales

- Presupuesto automático según ingresos y perfil (Estudiante / Estudiante-Trabajador / Trabajador)
- Registro rápido de gastos con categorías
- Cálculo de **costo de oportunidad** y **horas de trabajo equivalentes**
- Metas de ahorro con seguimiento
- Sistema de **rachas (streaks)**
- Ranking de amigos en tiempo real
- Dashboard con gráficos visuales
- Soporte offline
- Notificaciones inteligentes
- Exportación de reportes (PDF/CSV)

---

## 🛠️ Tecnologías Utilizadas

- **Flutter** + **Dart**
- **Firebase** (Authentication + Firestore)
- **Riverpod** (State Management)
- **GoRouter** (Navegación)
- **Drift** (Base de datos local SQLite)
- **fl_chart** (Gráficos)

---

## 📱 Requisitos

- Flutter 3.24 o superior
- Dart 3.5 o superior
- Cuenta de Firebase configurada
